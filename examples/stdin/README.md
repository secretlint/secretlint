# example-cli

An example for secretlint cli.

## Install

    yarn install
    
## Usage

```
$ yarn secretlint "**/*"
```

## Changelog

See [Releases page](https://github.com/secretlint/secretlint/releases).

## Running tests

Install devDependencies and Run `npm test`:

    npm test
